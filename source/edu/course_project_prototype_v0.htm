Рендеринг резкости | План и прототип

Инструментарий:

  CSS3 + HTML5 any gaming framework + set library javascript

Концепт:

 - Поверхность писменного рабочего стола с обьектами на нём. Панорамный просмотр и воздействие мышью на обьекты и обработка клавиатуры.

  • фокус на обьекте по наведению
  • параллакс эффект на скролл мышки
  • анимация движения обьектов
  • перемещение обьектов
